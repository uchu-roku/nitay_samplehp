# nitay株式会社 ホームページ

北海道から宇宙産業の裾野を広げる、nitay株式会社のコーポレートサイトです。

## 特徴

- **モダンなデザイン**: 白を基調にグリーンをアクセントカラーとした洗練されたデザイン
- **レスポンシブ対応**: スマートフォン、タブレット、デスクトップに最適化
- **SEO最適化**: 構造化データ（JSON-LD）実装済み
- **スムーズなアニメーション**: スクロールに応じたフェードイン効果

## ファイル構成

```
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptファイル
└── README.md       # このファイル
```

## セクション構成

1. **ヒーロー**: 企業のメインメッセージと創業情報
2. **会社紹介**: ミッション、バリュー、名前の由来
3. **事業内容**: 3つの主要サービス（林業支援、人材確保・育成、検証・試作）
4. **代表メッセージ**: 代表取締役社長の経歴とメッセージ
5. **入札・選定実績**: 実績の紹介
6. **会社概要**: 基本情報
7. **お問い合わせ**: コンタクトフォーム

## カスタマイズ方法

### コンテンツ編集
`index.html`の各セクションを直接編集してください。

### フォーム送信先の設定
`script.js`のフォーム送信ハンドラーを編集し、実際のサーバーエンドポイントに接続してください。
